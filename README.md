# GNN-MultiOmics 

### Project Title
**Graph Neural Network-Based Multi-Omics Integration for Predicting Cancer Drug Response**

## Project Overview
This project aims to develop a Graph Neural Network (GNN) framework to integrate multi-omics data (Genomics, Transcriptomics, etc.) to improve the accuracy of cancer drug response predictions. This is a Final Year Project (FYP) conducted at Monash University.
